## About

> [!NOTE]  
> **Contributions are welcome.** If you want to help improve the application, optimize the tracking, or add new features, feel free to fork the repository and submit a pull request.

FRAPP acts as a centralized messenger for deal-hunting. It monitors platforms like GOG, itch.io, Epic Games Store, and Steam, ensuring you can expand your library without scouring multiple storefronts daily.

---

## Deployment

This is a mobile application built with [Expo](https://expo.dev) using [`create-expo-app`](https://www.npmjs.com/package/create-expo-app). 

To get started, fork this repository to your own GitHub account and clone it to your machine.

## Running Locally

Follow these steps to set up the development environment:

1. **Clone the project**
   ```bash
   git clone [https://github.com/Brianali-codes/Frapp.git](https://github.com/Brianali-codes/Frapp.git)
   cd Frapp
  
2. **Go to the project directory**

  ```bash
  cd Frapp
  ```

3. **Install dependencies and Expo cli globally or Locally**

  ```bash
    npm install
  ```
4. **Install Expo CLi globally.**
  ```bash
    npm i -g expo
  ```
5. **Install Expo CLi to project only.**
  ```bash
    npm i expo
  ```
6. **Start the server for web**
  ```bash
    npm run web
  ```
7. **Start the server using expo.**
  ```bash
    expo start
  ```
